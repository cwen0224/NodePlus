節點編輯器

快捷鍵說明

  • Ctrl/Cmd + N在視窗中心新增節點
  • Ctrl/Cmd + A全選節點
  • Ctrl/Cmd + 點擊切換節點多選
  • Ctrl/Cmd + C / X / V複製 / 剪下 / 貼上
  • Ctrl/Cmd + D複製一份
  • Delete / Backspace刪除選取節點（或 hover 連線）
  • Ctrl/Cmd + Z復原
  • Ctrl/Cmd + Shift + Z重做
  • Ctrl/Cmd + Y重做（替代）
  • Ctrl/Cmd + +/-/0放大 / 縮小 / 重設縮放
  • F聚焦目前選取或全部內容
  • 方向鍵移動選取節點（Shift 為大步距）
  • Ctrl/Cmd + 拖曳套索框選
  • 右鍵開啟功能選單
  • Esc關閉面板或取消目前操作
